# zhouxiaotian_noc

This repository currently includes a filtered source snapshot of the local BookSim study workspace.

Included in `zhouxiaotian_noc_source_snapshot.zip`:
- source code under `src/`
- docs under `doc/`
- experiment scripts under `tests/scripts/`
- input configs under `tests/data/inputs/`
- helper scripts and runfiles

Excluded from the snapshot:
- generated build artifacts
- generated experiment results under `tests/data/results/`
- logs, temp configs, caches, and `src/booksim.exe`

The local workspace also contains untracked/generated result folders that were intentionally not uploaded.
